### 🧠 Concept: The Protocol of Release

**EN:** A departure from the 150-year-old QWERTY layout. An interface aiming for "direct output of thought" through 5-bit chords and finger independence. This is not a tool for competing in speed. It is a protocol to find your own "5-bit hole" and record the explosion of life.

**JP:** 150年前のQWERTYからの脱却。5ビットの和音（Chord）と「指の独立性」を鍵とした、思考の直接出力を目指すインターフェース。これは速度を競う道具ではない。自分だけの「5ビット・ホール」を見つけ、生命の爆発を記録するためのプロトコルである。

### 🛠 Standard Layout v1.8 (Master Edition)

"Thumb = e" axis. A professional layout utilizing anatomical finger independence.  
「親指 = e」を軸とし、解剖学的な指の独立性を利用したプロフェッショナル配列。

#### 1. Core Vowels (1-Bit)

The most agile finger, the thumb, is assigned to the vital `e`.  
最も動かしやすい親指に、最頻出の `e` を配置。

| Chord (Bit) | Char  | Finger              |
| :---------- | :---: | :------------------ |
| **16**      | **e** | **Thumb (Primary)** |
| **8**       | **a** | Index               |
| **4**       | **i** | Middle              |
| **2**       | **u** | Ring                |
| **1**       | **o** | Pinky               |

#### 2. Advanced Isolation (Specific Chords)

Chords output by "relaxing" specific fingers—a feat of high-level physical control.  
特定の指を「脱力（Isolation）」させることで出力される高度なコード。

| Chord (Bit)       |   Char   | Description                                            |
| :---------------- | :------: | :----------------------------------------------------- |
| **29 (16+8+4+1)** |  **q**   | **Master's Gate (Ring finger isolation / 薬指浮かし)** |
| **15 (8+4+2+1)**  |  **BS**  | **BackSpace (Thumb isolation / 親指浮かし)**           |
| **23 (16+4+2+1)** | **[SP]** | Space (Index finger isolation / 人差し指浮かし)        |
| **31 (All)**      |  **\n**  | **Total Release (Enter / 全押し)**                     |

#### 3. The Tras (Layer Switch)

Hold **24 (Thumb + Index)** for **500ms** to toggle the world (Layer).  
親指と人差し指（24）を **500ms長押し** して離すことで、次元（レイヤー）を切り替えます。

- **ABC MODE:** Alphabet and basic symbols (Default)
- **NUM MODE:** Numbers (16=0, 8=1...) and symbols (!, ?)
